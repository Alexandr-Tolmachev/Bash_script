Данный bash-скрипт, будет бэкапить все файлы с кодом, которые лежат не выше домашнего каталога текущего пользователя.

В качестве аргументов скрипту передается название каталога, название конечного tar-архива и расширения файлов (не обязательно одно, их может быть несколько, записанных через пробелы). 

Первый аргумент – название каталога
Второй аргумент – название архива
Третий и последующий аргументы (если они есть) – желаемые расширения

Пример:

bash script.sh /home/Alex/ back_up cpp py sh   - результатом такого запроса будет tar-архив back_up.tar, в котором будут содержаться ВСЕ файлы, которые находятся не выше каталога /home/Alex/ и имеют расширения cpp, py или h.
